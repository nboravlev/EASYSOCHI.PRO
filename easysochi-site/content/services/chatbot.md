## Возможности

- Регистрация и идентификация пользователей;
- Проверка адресов и геолокация MapBox;
- Алгоритм подбора и заказа товара/услуги;
- Календарь занятых дат для собственника;
- Проверка доступности товара для продавца;
- Личный кабинет собственника/продавца и арендатора;
- Мгновенные уведомления;
- возможна интеграция с доставкой (СДЭК или собственная разработка);
- Админский веб-интерфейс;

## Технологии

- **Framework**: Python (Aiogram,python telegram bot)
- **Database**: PostgreSQL
- **Maps**: Mapbox API
- **Payments**: Yandex.Kassa/Юмани/Payture
- **Deployment**: Docker + Nginx

## Деплой

Все продукты контейнеризованы в Docker и разворачиваются на продуктовых серверах в 99.9% доступностью.
